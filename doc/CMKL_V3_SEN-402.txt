Details for Competency SEN-402 (Software Project Management)

IDENTITY CODE: SW_PROJ_MGMT_4
REQUIRED? false
CREDITS: 4 (Graded)
DESCRIPTION
This competency introduces students to the concepts and tools in "classical" software project management: task breakdowns, task dependencies, critical path method, resource allocation issues and so on. Upon completing the competency, a student will be equipped to plan and manage a small project.
RESPONSIBLE INSTRUCTOR: Sally Goldin
