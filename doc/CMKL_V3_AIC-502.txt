Details for Competency AIC-502 (Reinforcement Learning)

IDENTITY CODE: REINFORCEMENT_ML_4
REQUIRED? false
CREDITS: 4 (Graded)
DESCRIPTION
Reinforcement learning is a branch of machine learning that operates using the principle of rewards for desired behavior. This competency explores the general concepts behind reinforcement learning, as well as several techniques and architectures for building RL systems.
