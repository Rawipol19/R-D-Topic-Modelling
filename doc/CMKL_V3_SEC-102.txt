Details for Competency SEC-102 (Data Reduction and Compression)

IDENTITY CODE: DATA_COMPRESS_4
REQUIRED? false
CREDITS: 4 (Graded)
DESCRIPTION
The size of data is the key for efficient communications. You cannot send a video of 10GB to 10,000 users in the network at the same time. In this competency, you will learn how to reduce the amount of data to be properly and efficiently collected, transmitted, and processed, using various compression techniques.
