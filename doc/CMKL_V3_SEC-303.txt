Details for Competency SEC-303 (Vulnerability Assessment for Software Applications)

IDENTITY CODE: VULNERABILTY_ASSMT_4
REQUIRED? false
CREDITS: 4 (Graded)
DESCRIPTION
Security vulnerabilities impact various software applications and may be harmful to users' information. Consequently, these vulnerabilities must be properly identified, assessed, and mitigated. In this competency, we will delve into details on techniques to assess security vulnerabilities for software applications, both manually and automatically.
