Details for Competency SEC-101 (Data and Information Fundamentals)

IDENTITY CODE: DATA_AQUISITION_4
REQUIRED? true
CREDITS: 2 (Graded)
DESCRIPTION
Data is the most valuable asset in modern information systems. How engineers properly and securely acquire, prepare, transform, and cleanse data objects, based on their characteristics, is a mandatory skill. In this competency, you will learn about data representation and types, how we can acquire data in the required format, and how we can handle data outliners.
